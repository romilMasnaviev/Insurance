TODO**Что было сделано и как? (краткое описание)**
Gradle представляет собой средство для сборки проекта. Написано на языке Groovy,
но представляет собой библиотеку для Java. Именно поэтому мы указываем в настройках
IDE версию Java для него. В папке gradle в корневой директории проекта есть
.jar файл - он и является библиотекой. Файл .properties - это настройки Gradle.

TODO**Зачем это было сделано? Чему научился в процессе выполнения задания?**
Сделал это для того, чтобы настроить базу для проекта, указать что моя директория
является самостоятельным модулем, который можно запускать отдельно.
