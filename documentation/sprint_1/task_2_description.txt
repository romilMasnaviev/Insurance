Команда build в Gradle
> Task :student_romil_masnaviev:compileJava - компилирует исходный код (main/java)
в .class файлы
> Task :student_romil_masnaviev:processResources - копирует ресурсы (main/resources)
> Task :student_romil_masnaviev:classes - объединяет предыдущие два пункта
> Task :student_romil_masnaviev:jar - упаковывает в .jar архив
> Task :student_romil_masnaviev:assemble - объединяет задачи сборки и готовит проект
для создания артефактов
> Task :student_romil_masnaviev:compileTestJava - компилирует тесты
> Task :student_romil_masnaviev:processTestResources NO-SOURCE - копирует ресурсы (test/resources)
> Task :student_romil_masnaviev:testClasses - объединяет предыдущие два пункта
> Task :student_romil_masnaviev:test - запускает тесты
> Task :student_romil_masnaviev:check - дополнительные проверки
> Task :student_romil_masnaviev:build - сборка, проверка, упаковка итогового артефакта
