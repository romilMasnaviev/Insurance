Реляционная БД - БД, основанная на реляционной модели данных,
данные организованы в виде связанных таблиц
СУБД - система управления базой данных
SQL - structured query language - язы структурированных запросов.
CREATE TABLE для создания таблицы в БД
CRUD операции - основные операции в DQL (Data Query Language)
Связи между таблицами - one/many one/one many/many
PrimaryKey - уникальный ключ для данной таблицы
ForeignKey внешний ключ - уникальный ключ из другой таблицы в данной таблице
